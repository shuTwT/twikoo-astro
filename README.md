# twikoo

一个**简洁**、**安全**、**免费**的静态网站评论系统。<br>
A **simple**, **safe**, **free** comment system.  



## 简介

这是一个[twikoo](https://github.com/twikoojs/twikoo)的魔改版，目标适配astro，理论上vue项目也能使用

~~终极目标：打包体积<100kb~~

**注意**：该仓库仅包含魔改版的前端，后端依旧为原版

更多详情请查看[原仓库](https://github.com/twikoojs/twikoo)
